Release Notes
xTuple Time & Expense (xtte)
Version 2.3.0
May 4, 2015
==================================

This is the release of the 2.3.0 xTuple Time Expense package,, a free and open source extension package for xTuple ERP
(PostBooks, Standard, and Manufacturing Editions).

The Time & Expense package (xtte) is available under the same
Common Public Attribution License (CPAL) as the PostBooks Edition
of xTuple ERP (www.xtuple.com/CPAL). For more information on
xTuple's extension package architecture, please see
www.xtuple.org/developer-zone.

==================================

The following features and bug fixes have been added to the
package since 2.1.0. Additional detail for each item listed below
may be found on our community website (www.xtuple.org). 

Features:
* desk [num]

Bug Fixes:

* Incorrect handling of Currency on Time Sheet Item [19541]
* Blank Time and Expense sheets are displayed on selecting to Edit/View from the Time and Expense History screen [20367]
* DB log error is displayed on selecting to invoice a time sheet [20845]
* T&E time sheet item - only able to select the most recently created item [22472]
* Changes for new naming conventions [22507]
