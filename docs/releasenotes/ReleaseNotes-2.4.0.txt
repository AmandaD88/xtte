Release Notes
xTuple Time & Expense (xtte)
Version 2.4.0
February 8, 2016
==================================

This is the release of the 2.4.0 xTuple Time Expense package, a
free and open source extension package for xTuple ERP (PostBooks,
Standard, and Manufacturing Editions).

The Time & Expense package (xtte) is available under the same
Common Public Attribution License (CPAL) as the PostBooks Edition
of xTuple ERP (www.xtuple.com/CPAL). For more information on
xTuple's extension package architecture, please see
www.xtuple.org/developer-zone.

==================================

The following features and bug fixes have been added to the
package since 2.3.0. Additional detail for each item listed below
may be found on our community website (www.xtuple.org). 


Features:

* Create TE filter to manage only timesheets of direct reports (FM2) [15494]
* Allow re-open of Closed T&E Sheet (FM2) [15975]
* Add Hours Column to TE List [18345]
* Adding the notes field to a column on the time and expense sheets list screen [19513]

Bug Fixes:

* Selecting to open the project assigned to a Time and Expense sheet clears the Item Number [19116]
* inactive employees can enter T&E worksheets [24496]
* the teemp table in the time & expense package does not have a primary key [26193]
